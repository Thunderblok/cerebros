# Cerebros (Elixir / Axon NAS Prototype)\n\nThis repository contains the Elixir/Axon port of the Cerebros neural architecture search prototype. More modules (application supervision, CLI, results collection) can be imported as needed.\n\n## Structure\n- lib/cerebros/architecture/spec.ex - Architecture spec and random generation\n- lib/cerebros/connectivity/builder.ex - Deterministic connectivity generation\n- lib/cerebros/networks/builder.ex - Axon model builder / compiler\n- lib/cerebros/training/orchestrator.ex - Trial orchestration GenServer\n- lib/cerebros/training/trial_worker.ex - Single-trial worker GenServer\n- lib/cerebros/data/loader.ex - Synthetic + placeholder dataset loaders\n\n## Quick Start\n```bash\nmix deps.get\nMIX_ENV=dev iex -S mix\n"""\nimport Cerebros.Architecture.Spec\n# build / run orchestrator manually (application module forthcoming)\n"""\n```\n\n## License\nTBD\n
